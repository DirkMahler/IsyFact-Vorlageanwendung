== Maven

Dieser Abschnitt beschreibt Regeln, die sich auf den Umgang mit Apache Maven beziehen.

[[maven:MainArtifact]]
.Alle durch Maven erzeugten Artefakte, die nicht Tests oder reine POMs reprÃ¤sentieren, werden mit `Main` gekennzeichnet.
[source,cypher,role=concept]
----
MATCH
  (:Maven:Project)-[:CREATES]->(mainArtifact:Artifact)
WHERE NOT
  mainArtifact.type in [
    "test-jar",
    "pom"
  ]
SET
  mainArtifact:Main
RETURN
  mainArtifact as MainArtifact
----
